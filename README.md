# Student — лабораторные работы по React Native (ССП)

**Автор:** Kiryl Pashkevich  
**Email:** pakir2005@gmail.com

В этом репозитории собраны лабораторные работы **№12–20** по дисциплине/курсу ССП (React Native).
Каждая лабораторная лежит в отдельной папке `Student_SPS_XX`.

## Технологии

- **JavaScript (React / React Native)**
- **Expo / Expo Snack** (для запуска на телефоне через Expo Go и QR-код)
- **React Hooks**: `useState`, `useEffect`, `useRef`, `useMemo`
- **React Native компоненты**: `View`, `Text`, `Image`, `TextInput`, `ScrollView`, `FlatList`, `Button`, `TouchableOpacity`, `SafeAreaView`, `StatusBar`, `Modal`, `Alert`
- **Стилизация**: `StyleSheet.create`, массивы стилей, flexbox, фиксированные/процентные размеры
- **Навигация** (в части работ): `@react-navigation/native`, `@react-navigation/native-stack`
- **Анимации**: `Animated` (`Animated.Value`, `timing`, `sequence`, `parallel`, `interpolate`)
- **Системные API**: `Share`
- **Иконки**: `@expo/vector-icons` (в отдельных работах)

## Содержание лабораторных

### Лабораторная №12 — подготовка Expo и первое приложение
Папка: `Student_SPS_12`

- **Тема**: настройка среды Expo (Snack/Expo Go) и вывод на экран `Hello World`.
- **Ключевые элементы**: функциональный компонент, `StyleSheet`, базовые `View`/`Text`.

### Лабораторная №13 — стили в React Native
Папка: `Student_SPS_13`

- **Тема**: стилизация интерфейса через `style` и `StyleSheet`, приоритет/наследование через массивы стилей.
- **Ключевые элементы**: оформление контента, несколько стилей для текста (`bigBlue`, `red`) и их комбинации.

### Лабораторная №14 — гибкие элементы (height/width, flex, проценты)
Папка: `Student_SPS_14`

- **Тема**: фиксированные размеры (`width`/`height`), гибкие размеры (`flex`), процентные значения (`'80%'`).
- **Ключевые элементы**: блоки с фиксированными размерами, строка с распределением по `flex`, блок с процентной шириной.

### Лабораторная №15 — основные компоненты React Native
Папка: `Student_SPS_15`

- **Тема**: использование базовых компонентов и построение осмысленного экрана(ов).
- **Ключевые элементы**: `SafeAreaView`, `ScrollView`, `View`, `Text`, `Image`, `TextInput`, `Button`, `Alert`.
- **Реализация**: два экрана (горизонтальный pager) с карточками, текстами, вводом email и кнопкой действия.

### Лабораторная №16 — навигация в React Native
Папка: `Student_SPS_16`

- **Тема**: многоэкранное приложение и переходы между экранами.
- **Ключевые элементы**: React Navigation (`NavigationContainer`, стек), передача параметров.
- **Реализация**: мини‑приложение “Книга контактов” — список, добавление, просмотр контакта.

### Лабораторная №17 — анимация в React Native
Папка: `Student_SPS_17`

- **Тема**: анимации через библиотеку `Animated`.
- **Ключевые элементы**: `Animated.Value`, `Animated.timing`, `interpolate`, анимированные изображения/элементы.
- **Реализация**: welcome‑слайдер с индикаторами страниц + простой splash‑экран.

### Лабораторная №18 — форма регистрации и авторизации
Папка: `Student_SPS_18`

- **Тема**: экран регистрации и альтернативный экран авторизации, валидация, сообщения об успехе.
- **Ключевые элементы**: `TextInput`, `TouchableOpacity`, `Alert`, переключение режимов “login/register”.

### Лабораторная №19 — среда разработки и передача данных с формы
Папка: `Student_SPS_19`

- **Тема**: работа с формой и передачей/использованием введённых данных.
- **Ключевые элементы**: состояние формы (`useState`), обработчик “submit”, вывод результата на экран.

### Лабораторная №20 — мобильное приложение “Социальная сеть”
Папка: `Student_SPS_20`

- **Тема**: лента постов, экран поста, профиль, полноэкранные изображения, кнопка “поделиться”.
- **Ключевые элементы**: `FlatList`, `Modal`, `Share`, работа с несколькими экранами (внутренняя маршрутизация по состоянию).
- **Реализация**: лента → детальный просмотр → профиль автора + подписки/лайки.

## Запуск (общая рекомендация)

В большинстве папок достаточно открыть проект в Expo/React Native окружении:

- **Expo**:
  - `npm install`
  - `npx expo start`

Для работ, сделанных в стиле Snack, достаточно открыть проект в **Expo Snack** и вставить соответствующий файл (`main.js`/`App.js`).
