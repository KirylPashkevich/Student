Лабораторная работа № 16. Навигация в React Native

Автор: Kiryl Pashkevich
Email: pakir2005@gmail.com

Цель работы:
- изучить навигацию между экранами в React Native;
- закрепить работу с экраном списка, экраном добавления и экраном просмотра данных.

Краткое содержание выполненной работы:
- реализовано приложение "Книга контактов";
- добавлены экраны списка контактов, добавления контакта и просмотра деталей контакта;
- выполнены переходы между экранами с передачей параметров;
- организовано хранение контактов через контекст (`ContactsContext`).

Использованные компоненты:
- `View`, `Text`, `FlatList`, `TouchableOpacity`, `TextInput`;
- `KeyboardAvoidingView`, `Alert`;
- `NavigationContainer`, `createNativeStackNavigator`.

Ответы на вопросы для контроля:
1) Для установки зависимостей Expo-проекта используется `expo install`.
2) React Navigation - библиотека для организации навигации между экранами.
3) Навигация нужна для переходов между экранами и передачи параметров.
4) Корневой контейнер навигации - `NavigationContainer`.
5) Основные типы навигации: Stack, Tab, Drawer (и их комбинации).
