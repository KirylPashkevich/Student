## Установка и запуск

Для управляемого проекта Expo зависимости уже указаны в `package.json`. Установите их:

```bash
npm install
npx expo install react-native-screens react-native-safe-area-context
```

Запуск:

```bash
npx expo start
```

Дальше откройте приложение на устройстве/эмуляторе (сканирование QR-кода или выбор платформы в терминале).

> Конфигурация Expo не требует кастомных изображений; при необходимости иконку и заставку можно добавить в `app.json` и папку `assets`.

### Ошибка `ENOENT: mkdir '...\node:sea'` на Windows

Возникает из‑за несовместимости Node.js v24+ с Metro на Windows. **Решение:** использовать Node.js v20 LTS.

1. Проверить версию: `node -v`
2. Установить Node.js 20 LTS (с [nodejs.org](https://nodejs.org) или через `nvm`)
3. Удалить папку `.expo` и перезапустить:  
   `rmdir /s /q .expo`  
   `npx expo start`

## Ответы на вопросы для контроля

**1. Как установить зависимости для управляемого проекта Expo?**

Через `expo install`: 
```bash
npx expo install react-native-screens react-native-safe-area-context
```
Так подбираются версии, совместимые с текущей версией Expo.

**2. Что такое React Navigation?**

React Navigation — это библиотека для организации навигации в приложениях на React и React Native. Она предоставляет навигаторы (стек, табы, drawer и др.) и единый подход к переходам между экранами, заголовкам и параметрам маршрутов.

**3. Для чего используется навигация?**

Навигация используется для переключения между экранами приложения, передачи параметров при переходах, организации истории переходов (кнопка «Назад») и структурирования приложения из нескольких экранов.

**4. Во что оборачивают приложение?**

Приложение оборачивают в компонент **`NavigationContainer`** из `@react-navigation/native`. Он должен быть корневым для всех навигаторов и хранит состояние навигации.

**5. Какие типы навигации можно сделать в приложении на React Native?**

- **Stack (стек)** — экраны друг над другом, переход вперёд/назад (`@react-navigation/native-stack`).
- **Tab (вкладки)** — переключение экранов внизу или сверху (`@react-navigation/bottom-tabs`, `@react-navigation/material-top-tabs`).
- **Drawer (боковое меню)** — выезжающая панель с экранами (`@react-navigation/drawer`).
- Комбинации: например, табы с несколькими стеками внутри каждой вкладки.
