Автор: Kiryl Pashkevich

Цель работы:
- получить навыки использования основных компонентов React Native;
- реализовать экран(ы) по заданию с применением базовых UI-элементов.

Краткое описание выполненной работы:
- создано приложение с двумя экранами, переключаемыми горизонтальным ScrollView;
- на первом экране реализованы карточки направлений, поле ввода email и кнопка действия;
- на втором экране реализованы статусы рейса, изображение, индикаторы страниц и кнопка действия;
- использованы компоненты `StyleSheet`, `Text`, `SafeAreaView`, `ScrollView`, `View`, `TextInput`, `Image`, `Button`, `Alert`.

Ответы на вопросы для контроля:
1) Базовый компонент - `View`.
2) Пользовательские компоненты интерфейса - `Button`, `Switch` и другие UI-компоненты.
3) Для Android-специфичных возможностей используются, например, `BackHandler`, `DrawerLayoutAndroid`, `PermissionsAndroid`, `ToastAndroid`.
4) Представления списка - `FlatList` и `SectionList`.
