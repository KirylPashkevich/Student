Лабораторная работа № 17. Анимация в React Native

Автор: Kiryl Pashkevich
Email: pakir2005@gmail.com

Цель работы:
- изучить базовые анимации в React Native;
- реализовать welcome-экран с анимированными элементами.

Краткое описание:
- реализован слайдер из нескольких экранов на `Animated.FlatList`;
- добавлены анимации прозрачности, масштаба и интерполяции;
- добавлены индикаторы страниц и кнопка действия;
- реализован простой splash-экран с последовательной анимацией.

Использованные инструменты:
- `Animated.Value`, `Animated.timing`, `Animated.sequence`, `Animated.parallel`;
- `interpolate`, `Animated.event`, `useNativeDriver`.
